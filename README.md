## Irises
Customizing the famous Iris Dataset in "Flowers and Numbers":
- A scatter plot with just one dot. We all have a starting point - flowers, humans and machines.
- machines start to count from 0, humans start to count from 1
- displaying the Iris Chrysographes with the most beautiful color
- sepals and petals, 4 measurements for each flower
- scatter plots as subplots
